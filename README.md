# City Creek Creations Website

This website uses the approved full-page homepage design.

## Upload to GitHub

Upload both:

```
index.html
assets/
  city-creek-homepage.png
```

Your repository should look like:

```
index.html
assets/city-creek-homepage.png
```

Then commit the changes.

## GitHub Pages

If GitHub Pages is already enabled, your site should update within 1-2 minutes.

## Important

Do not rename the image unless you also update the image path in `index.html`.
